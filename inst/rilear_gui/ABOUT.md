### About rilear

This web application is a front-end for the [R](https://www.r-project.org/) package [`rilear`](https://github.com/dwoll/rilear). The package provides functionality for assessing the lifetime excess absolute risk from radiation exposure. Based on models for excess relative and excess absolute cancer risk for external low-LET radiation exposure. Risk measures include LAR / LEAR / CER, REID / REIC, ELR, and RADS. Supports multiple exposure events and simultaneous application to a whole population to project total number of expected cancer cases.
